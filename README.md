# F1 Pick'em

A full-stack website for making Formula 1 picks with your friends.

Check it out at www.f1pickem.com

### Technologies

- Go
- AWS Lambda
- AWS API Gateway
- AWS DynamoDB
- Javascript
- React
- Redux
- AWS Amplify
- AWS Route 53
- [Ergast Developer API](http://ergast.com/mrd/)


## Development

```
# backend
cd backend
make api

# frontend
cd web
npm run start
```
